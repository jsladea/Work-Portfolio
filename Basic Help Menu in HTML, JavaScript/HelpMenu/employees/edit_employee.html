Edit an Employee

The Edit Employee page can be accessed via the "Administrative" section of the "Management Tools" toolbar under the "Edit" dropdown.



Instructions:

Editing an Employee

  1. Navigate to the Edit Employee page.
  2. Select an employee to edit and press the "Search" button.
  3. You can modify their department, Email, or Access level. Note: Future support is coming for the update of Job Titles as soon as standardized titles become used.
  4. Press the "Save Changes" button.


Reseting an Employee's Password:

This is only necessary if an employee can not remember the answers to their security questions. Note: This will only reset an employee's uLearn password if they are not using windows login. If an employee has a windows login, then they will need to contact the IT department to reset their password.

  1. Navigate to the Edit Employee page.
  2. Select an employee and press the "Search" button.
  3. Press the "Reset Employee ULearn Password" button.
  4. Give the code to the employee so that they can reset their password.
  5. The employee can reset their password from the login screen of the uLearn application.

Change an Employee's Username

If an employee changes their windows login username with the IT department, you will need to update it in the uLearn application as well. This will work for both employees using windows login and employees using the uLearn login system.

  1. Navigate to the Edit Employee page.
  2. Select an employee and press the "Search" button.
  3. Enter the desired new username for the employee.
  4. Confirm the new username by entering it again.
  5. Press the "Change Username" button.
