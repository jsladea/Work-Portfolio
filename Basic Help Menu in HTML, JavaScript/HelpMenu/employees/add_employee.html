Add an Employee

The "New Employee" button can be found in the "Administrative" section of the "Management Tools" toolbar under the "Add" dropdown.


Instructions:

Adding a New Employee

  1. Navigate to the Add New Employee page.
  2. Enter a username for the employee. If the employee already has a windows login (for the computers at UST), enter their windows login username so they don't have to remember a new username and password.
  3. Enter their first and last names
  4. Select a Department.
  5. Enter a Job Title for the employee.
  6. If the employee has a UST email address, enter it. If not, leave blank.
  7. Select the rights you wish the user to have in the uLearn application.
